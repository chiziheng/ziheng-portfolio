This is a 3 people group project. The goal is to train a model that can predict tweet sentiment using training data. The overall process consists EDA, data preprocessing, model training & selection, hyperparameter tunning and prediction. The final chosen model is a neural network model.

Some technics used during text preprocessing:

  • Remove links
  • Remove mentions
  • Replace tags
  • Replace ordinal
  • Replace textical emoticon
  • Replace word contraction
  • Remove punctuation
  • Stem words
  • Create document term matrix
  • Remove sparse terms
  • One hot encoding
