This project is done in collaboration with AXA Investment Manager. We were provided with the China A-shares related data and tasked building a predictive model that can serve as a low-volatility indicator. The project cycle consists problem scoping & definition, data cleaning & analysis, model training & evaluation

The final chosen model is ARIMA as this is a time-series problem in nature and ARIMA excels in performance by taking patterns in the past, rate of data changes, and noise between consecutive data into consideration

For more details, check the project source and poster

Technologies Used

  • Python
  • Pandas
  • Matplotlib
  • ARIMA
