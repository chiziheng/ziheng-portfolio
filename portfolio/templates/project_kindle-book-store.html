This is a group project completed by 4 people. The end product is a kindle book online review website. The website is built from end-to-end and consisting both production and analytics systems. Production system contains web server and two databases while analytics system uses Hadoop cluster to calculate Pearson Correlation and TFIDF. The deployment and tear down of the website is fully automated using script so anyone can auto deploy it with AWS credentials.

My main responsibility in this project is taking care of the databases (Mysql and Mongodb) and I used bash and sql scripts to make sure they can be auto deployed and loaded with all the required data.

Feel free to check the slides as it contains many details and you are welcome to fork the repo and play it yourself!

Technologies Used

  • AWS
  • Bash
  • SQL
  • MYSQL & MongoDB
